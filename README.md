
# Real-Time Cryptocurrency Tracker

🚀 **Live Demo**: [Real-Time Cryptocurrency Tracker](https://ramesh-kumawat.github.io/Real-Time-Cryptocurrency-Tracker/)

## 📖 Overview

The **Real-Time Cryptocurrency Tracker** is a web application designed to track cryptocurrency prices in real-time. It provides an easy-to-use interface for users to monitor price changes and stay informed about the cryptocurrency market. The application is built with a focus on speed, reliability, and responsiveness.

## ✨ Features

- **Live Price Updates**: Displays real-time price updates for various cryptocurrencies.
- **Search Functionality**: Allows users to search for specific cryptocurrencies.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Clean and Intuitive UI**: A user-friendly design to enhance user experience.
- **Performance Optimized**: Built with Vite for fast development and deployment.

## 🛠️ Technologies Used

- **Frontend Framework**: [Vite](https://vitejs.dev/) for fast builds and hot module replacement.
- **JavaScript**: For dynamic functionalities and handling API responses.
- **HTML5 & CSS3**: For creating a structured and visually appealing interface.
- **External APIs**: Integrated for fetching live cryptocurrency data.

## 📦 How to Run Locally

Follow these steps to run the application locally on your machine:

1. Clone the repository:

   ```bash
   git clone https://github.com/Ramesh-kumawat/Real-Time-Cryptocurrency-Tracker.git
   cd Real-Time-Cryptocurrency-Tracker
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to:

   ```
   http://localhost:3000
   ```

## 🚀 Deployment

The app is deployed using **GitHub Pages**. To deploy your own version:

1. Build the app:

   ```bash
   npm run build
   ```

2. Deploy the `dist` folder using the following command:

   ```bash
   npm run deploy
   ```
